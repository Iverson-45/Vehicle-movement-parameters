# Vehicle-movement-parameters

The system calcualates and displays:
---------------------------------

Speed [km/h]

Acceleration [m/s^2]

Power [Hp]

Torque [Nm]

Inclination[°]


Hardware:
----------------------

Microcontroller: STM32F303K8

GPS module: U-blox NEO-M8N

IMU + GYRO: BMI160

Display: 2.54 inch LCD 128x64 ST7567s

Stages:
--------

Display speed
